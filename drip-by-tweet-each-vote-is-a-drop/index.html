2014 – Drip-By-Tweet: Each Vote is a Drop

 

Team of developers Domestic Data Streamers created a real-time physical visualization of votes for a graphic design contest. A person can vote for a specific piece by sending a tweet, after which the machine releases a drop of yellow liquid and sends it to the corresponding test tube.

Source: Domestic Data Streamers.
